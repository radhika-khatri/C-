# 💻 C++ Lab Exercises

A collection of C++ programs written as part of lab coursework. Each folder corresponds to a lab exercise covering a different concept in C++, from basic syntax and control flow all the way through object-oriented programming and data structures.

If you're learning C++ or just want to look back at foundational exercises, this is a clean, organized reference.

## 🧪 What's Covered

The exercises progressively build on each other, typically covering topics like:

- Basic input/output and variables
- Conditional statements and loops
- Functions and recursion
- Arrays and strings
- Pointers and memory management
- Object-oriented programming (classes, constructors, inheritance)
- File handling
- Data structures (linked lists, stacks, etc.)

## 🚀 How to Compile and Run

You'll need a C++ compiler like `g++` installed.

### On Linux / macOS

```bash
g++ Lab_Ex_1/filename.cpp -o output
./output
```

### On Windows

Use any of the following:
- **MinGW** with the same `g++` command in terminal
- **Visual Studio** (open the `.cpp` file and run directly)
- **Code::Blocks** or **Dev-C++**

### Or use an online compiler

If you don't want to set anything up locally, paste the code into [OnlineGDB](https://www.onlinegdb.com/) or [Replit](https://replit.com/) and run it there.
